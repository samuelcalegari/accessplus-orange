Help about the User Confort+ service

User Confort+ enables you to adjust display and behaviour settings on web pages you are reading.

Quick settings

Font size

This setting allows you to choose a font size magnification.

Disable colours

Displays the page in black and white (or with the colours you have set in advanced settings).

To return to the original colours of your browser, you can click on this button again.

Reading guide

Displays a ruler or a mask for easy reading.

More settings

Displays the full settings panel.

Profiles

Allows activation of parameters you have set and saved or to choose among preset profiles:

  • Improve readability (enlarges text, changes colours, and replaces the site’s fonts with Arial)
  • Make reading easier (replaces all fonts of the site with Arial, enlarges the text and space between lines, aligns the texts on the left, adds a number to the elements composing lists on the page and displays the reading mask )
  • Navigate effortlessly (displays a remote control to scroll in the page and lets you navigate by hovering over page elements)

Full settings

Typography

“Font size”

This setting allows you to enlarge the site’s font size.

“Font face”

Allows you to modify the font face defined by the web site to improve the reading confort. For instance you can choose “Arial” and “Open Dyslexic”.

“Text case”

Allows you to modify texts display according to your needs: by capitalizing the first letter of each word, to lowercase or to uppercase all text.

Reading help

“Space between lines”

Allows you to increase the space between lines to ease reading.

“Space between words”

This setting allows you to add space between each word in texts.

“Reading mask settings”

Allows you to display a reading ruler, vertical or horizontal, or to display a more or less transparent reading mask so that you can concentrate on the part of the text you are reading.

“Other settings”

“Align texts to the left”: Justified texts often disturb reading. This option allows to disable text justification and to align all texts to the left.

“List elements numbering”: Displays a numerical index in front of all list elements.

Colours

Allows to define a font colour and a background colour.

You can use either a combination of colours (“preset colour combination”) or redefine the setting (“personalized colour combination”).

To go back to the original colours of the browser, you can cancel all settings by clicking on the Colours pictogram in the Quick settings bar, or uncheck the setting in the Colours tab.

Behaviour

“Directly jump to content whenever it’s possible”

This setting allows an automatic positioning on main page content, in particular by jumping navigation links.

If the webmaster has not clearly marked up in the page code where the content is, this doesn’t work.

“Navigation on hover”

Displays a remote control with up and down arrows to scroll the page without clicking. When you hover over an element that you would want to click, just leave your mouse there and it clicks for you after a while (the delay you have defined).

“Automatic selection of elements”

Moves through all clickable elements in the page as well as all form elements. You just have to hit a key to stop it and show an actions menu. To go back to browsing the page elements, just hit another key.

Advanced settings

“Remove layout”

Replaces site fonts with your default fonts (those defined in your browser). Moreover, content is displayed in one column linearly.

“Remove transparency effects”

Enables the deactivation of possible transparency effects of the page, e.g. grey veils on disabled buttons. This minimizes disruption when reading content.

“Remove background images”

Removes background images that could disturb reading. Keeps the foreground information only.

“Remove foreground images”

Removes images contained in the page that could disturb reading. They are replaced by their text alternatives. A link allows to display the image on demand.

“Reset” button

Restores the site to its initial state, by removing all settings you could have choosen with Confort+.

Compatibility with Assistive Technologies

If you're using assistive technologies, please read this section.

Screen readers

  • If your screen reader is in “mouse cursor” mode, a simple mouse hover can trigger a link if you chose a motor help like “navigation on hover”.
  • The “automatic selection of elements” mode (looping through all clickable elements in the page) is very difficult to use with a screen reader, we don’t advise it.

Screen magnifiers

If you have configured your screen magnifier to alter colours in web pages, then the settings of Confort+ may be perceived incorrectly; or they may conflict with your own settings. If so, choose a combination like “keep the original colours”.

Voice control

We have found, at least with Dragon Naturally Speaking, that motor help modes are problematic. We advise you not to use them in conjunction with this software.

Troubleshooting

Cookies

The Confort+ service uses cookies, which are small text files that enable your browser to store your settings on your computer. If your browser doesn't accept cookies, then the service will not operate properly.

Javascript

The Confort+ service is a library distributed on the Orange Group websites, and it relies on Javascript code that modifies pages once they are loaded. If for one reason or another you have disabled Javascript (for example because of a technical restriction by your administrator), then the service will not operate properly.

Privacy

No data pertaining to your use of Confort+ is stored on our servers. All your settings are stored, for persistence purposes only, under the form of cookies, that you can destroy at any time by using the “Delete all settings” command. Please note, though, that for technical reasons the User confort service will only operate properly if cookies are enabled..
